Exercise 6

1. Teachers can be given a template for their own HTML page so that any changes specific to their class can be made without interfering with the shared structure of the website.
Style will be maintained and uniform through the use of an external stylesheet, which the teachers will not require access to.
Control over the stylesheet and other important files, such as the home page, will be delegated to an administrative staff member.
2. As stated in my response to prompt #1, teachers will be given a template for their own HTML page to implement informational changes specific to their class.
A single external stylesheet will be used to style the website. The teachers will not have access to this file. This single stylesheet will maintain the uniformity of the website.
Since this stylesheet will be external, any changes made to the stylesheet can be applied throughout the website without any additional effort from teachers.
Also, any changes made by the teachers to their own HTML file will not impact or alter the external stylesheet.
3. Please find the list of ten or more HTML tags and classes that may be used for the website below.

  1. table.assignments : a class for a table to show when assignments are due
  2. table.exams : a class for a table to show exam dates and times
  3. table.times : a class for a table to show class times
  4. table.subjects : a class for a table to show subjects taught; syllabi will be included in this table
  5. table.teachers : a class for a table that shows a directory of teachers with their contact information
  6. body.(subject) : i.e. body.history; a class to style the body for each subject ; choices for background color and font color belong here
  7. h1.(subject) : a class to style the h1 element for each subject ; choices for font family and weight belong here
  8. p.(subject) : a class to style the p element for each subject ; choices for font family and weight belong here to differentiate between a paragraph and a header
  9. link : links should be styled in the external stylesheet to ensure uniformity across the website ; NOTE: these will not conflict with background color as teachers have no control over this sheet
  10. ul, ol: lists (such as those for course expectations) should be styled in the external stylesheet to ensure uniformity across the website

4. Considerations for the CSS code to implement these stylistic requirements are as follows:

  • The table classes will be universal, meaning they will look the same between departments.
  • The body, header, and paragraph elements will differ between departments.
  • This means that a separate class will be created for each element for each department
  • The style for links and lists will be universal, meaning they will look the same between departments.
  • Teachers within each department will need to decide how they want the departmental style to look before the CSS code can be written.
